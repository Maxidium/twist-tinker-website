# Twist Tinker 
![Twist Tinker Preview](images/preview.png)

Адаптивна цільова сторінка для вигаданої консалтингової компанії з унікальним фірмовим стилем, гумористичним копірайтингом та сучасним фронтенд дизайном.

[Live Demo](https://maxidium.github.io/twist-tinker-website/) | [Figma](https://www.figma.com/design/bdxOt4JUwOcBweu7yMlrhp/Twist---Tinker?node-id=0-1&t=vwEWrnbHoU0LsEIT-1)

## Особливості
- Креативний вигаданий фірмовий стиль
- Адаптивний дизайн
- Семантична розмітка HTML5
- CSS-анімація та переходи
- Інтерактивна навігація

## Технології
- HTML5
- CSS3
- JavaScript
